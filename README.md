# schedule-notifier

## Requirements
* Python 2.7+ or 3.2+
* `requirements.txt`

## Usage
* Install `requirements.txt`
* Copy `conf.example.py` to `conf.py`
* Edit `conf.py` (You can obtain Twitter API tokens from https://apps.twitter.com)
* Run `app.py`
